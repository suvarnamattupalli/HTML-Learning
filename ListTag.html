List Tag

To arrange elements in order we use List.

Types of Tags

  1. Ordered List
  2. Unordered List
  3. Description List

Ordered List

  1. Elements stores in order.
  2. Using Numbers, Alphabets - (Capital,Small), Roman Numbers
  3. We use ol & li tag
  4. Types of Attributes to use in ordered list.
    1. type
    2. reversed
    3. start
