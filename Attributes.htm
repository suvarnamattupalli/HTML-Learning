Attributes

Attributes provide additional information about HTML elements. They are always included in the opening tag.

Attributes usually come in name/value pairs like name="value"

Common Attributes

  • style - Specifies an inline CSS style for an element
  • title - Specifies extra information about an element (displayed as a tooltip)
  • href - Specifies the URL of a link (used in <a> tag)
  • src - Specifies the path to an image (used in <img> tag)
  • alt - Provides alternative text for an image (used in <img> tag)
  • width and height - Specifies the width and height of an element (used in <img> and <canvas> tags)
  • target - Used in anchor tag to open link in new Tab
